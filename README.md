# go-roleplay
A set of libraries to make roleplaying games easier on golang.

## What this project aims to be
- A set of clean, simple to use, and flexible systems you can plug into RPGs that you develop in go
- Useful for hobbyists and commercial use
- Fully open source under the MIT license so you can use it in proprietary products

## Current modules being worked on
- Inventory
    - Basic item models, slots, stacking rules and inventory logic
    - Item creation as well

## Modules planned so far
- Items

## Usage
Install the module:
```sh
go get github.com/reneonthemaker/go-roleplay
```
Import a package:
```go
import "github.com/reneonthemaker/go-roleplay/inventory"
```
